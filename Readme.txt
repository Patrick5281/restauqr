Restaurant Digital Ordering System

Description
Ce projet vise à digitaliser les services rendus dans un restaurant en offrant une expérience client simplifiée et moderne via une mini-application accessible par QR code. Une fois le code scanné, les clients accèdent à une interface intuitive pour :

Explorer les informations générales du restaurant (localisation, section "À propos").

Consulter le menu, filtrer par type (entrée, plat, dessert) et accéder aux descriptions détaillées des plats.

Ajouter des plats au panier, choisir la quantité, et passer leur commande en indiquant leur numéro de table.

Recevoir une confirmation de commande avec une estimation du temps de préparation.


Fonctionnalités clés

Menu digital interactif avec filtres.

Panier intelligent avec gestion des quantités.

Notification du statut de commande.

Intégration d'une interface utilisateur moderne et intuitive 
 

Objectifs

Améliorer l'expérience client en restaurant.

Réduire les erreurs de commande et optimiser le service.

Moderniser l'image des restaurants avec une solution digitale simple et accessible.
